Lesson 1: An Explanation of HTML and CSS

What is HTML?

HTML stands for Hyper Text Markup Language

  1. It is a standard markup language for web page creation
  2. It defines the structure of web content
  3. It consists of a series of elements
  4. HTML element tell the browser how to structure of sections, paragraphs, and links of the content

What is CSS?

CSS stands for Cascading Style Sheets

  1. It describing the presentation of HTML elements
  2. Those presentations including colors, layout, and fonts
  3. There are internal, external and inline CSS styles
